Demand Random Routes

Edit on GitHub

DUAROUTER and JTRROUTER used to generate random routes for a given road network with the option (or for short). Due to a number of bugs in this feature and because those routes are highly unrealistic, this behavior was replaced with a python random trip generation script which can also produce a route set employing the DUAROUTER.
